# Linux-Notes
My Linux Learning notes and scripts
## 📋 Basic Commands
| Command | Description |
|---------|-------------|
| ls | List files |
| cd | Change directory |
| pwd | Show current path |
| mkdir | Create folder |
| rm -rf | Remove folder |
| chmod | Change permissions |
| grep | Search text |
| ps aux | Show processes |

## 🔧 Service Management
| Command | Description |
|---------|-------------|
| systemctl status | Check service |
| systemctl start | Start service |
| systemctl stop | Stop service |
| systemctl restart | Restart service |

## 📝 Log Management
| Command | Description |
|---------|-------------|
| journalctl -p err | Show errors only |
| journalctl -f | Real-time logs |
| tail -f /var/log/syslog | Watch syslog |

## 🌐 Network Commands
| Command | Description |
|---------|-------------|
| ip addr | Show IP address |
| ping | Test connection |
| netstat -tulpn | Show open ports |
| ssh user@host | Remote connect |
